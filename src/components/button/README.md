> 常用的操作按钮。

## Attributes 属性

参数 | 说明 | 类型 | 可选值 | 默认值
--- | --- | --- | --- | ---
size | button尺寸类型 | String | mini, small, medium, large | medium
type | button形式类型 | String | primary, success, warning, error, text, normal | normal
disabled | 是否禁用状态	 | boolean | - | false

## Events 事件

事件名称 | 说明 | 回调函数参数
--- | --- | --- | 
onClick | 当点击button时触发 | -
